sherror

IPW Shell-Support Command

NAME

sherror - standard IPW error message for shell scripts

SYNOPSIS

sherror pgm message [ file ]

DESCRIPTION

sherror is an error message generator that may be called by IPW shell scripts. The error messages are printed in a similar format to the IPW error message standard for C programs.

OPTIONS

None.

OPERANDS

pgm
Name of the script that generated the error.

message
Message explaining the error.

file
Name of file that error is associated with.

EXAMPLES

sherror $0 "$1: unsupported option"

sherror $0 "can't access file" $filename

FILES

None.

DIAGNOSTICS

Always exits with nonzero status.

RESTRICTIONS

None.
