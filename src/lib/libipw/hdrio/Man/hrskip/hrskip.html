hrskip

IPW Library Function
Library - libipw
Category - hdrio

NAME

hrskip - skip input header records

SYNOPSIS

#include "hdrio.h"

int
hrskip(
	int       fdi)		|* input file descriptor	 *|

DESCRIPTION

hrskip skips header records on file descriptor fdi, stopping after a preamble record has been read.

Call hrskip if you want to skip the (remainder of the) current input header without ingesting it or passing it to another image.

RETURN VALUE

OK for success, ERROR for failure
