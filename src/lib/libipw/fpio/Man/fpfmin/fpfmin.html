IPW Library Function
Library - libipw
Category - fpio

NAME

fpfmin - access fpio minimum fpixel values

SYNOPSIS

#include "fpio.h"

fpixel_t *
fpfmin(
	int      fd)		|* image file descriptor	 *|

DESCRIPTION

fpfmin accesses the array of per-band minimum fpixel values associated with file descriptor fd.

If fpfmin is called BEFORE any floating-point header I/O has occurred on fd, then a default pixel<->fpixel conversion will be established, which will be unaffected by subsequent floating-point header I/O.

IPW programs use fpfmin, in conjunction with fpmap, if they wish to modify the default conversion(s) from pixels to fpixels. Such programs typically read pixels with pvread, apply the map(s) explicitly, then write fpixels with fpvwrite.

RETURN VALUE

pointer to band-indexed array of minimum fpixel values, or NULL for error.
