fphdrs

IPW Library Function
Library - libipw
Category - fpio

NAME

fphdrs - read, then write any floating-point headers

SYNOPSIS

#include "fpio.h"

void
fphdrs(
	int     fdi,		|* input image file descriptor	*|
	int	nbands,		|* # image bands		*|
	int     fdo)		|* output image file descriptor	*|

DESCRIPTION

fphdrs copies all (remaining) image headers from filw descriptor fdi to file descriptor fdo. Any floating-point headers encountered that are known to fphdrs (currently: LQ) are also ingested, and thus become accessible to the fpio initialization routine.

Only the LQ header is recognized by this version of fphdrs.

fphdrs calls gethdrs; therefore, no further headers may be ingested after fphdrs is called. See gethdrs() for more info.

Call fphdrs if your program doesn't need to ingest any more headers, but you want the floating-point headers read in order to initialize the fpio package.

RETURN VALUE

None.
