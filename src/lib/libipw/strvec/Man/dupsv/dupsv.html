NAME

dupsv - duplicate a strvec

SYNOPSIS

#include "ipw.h"

STRVEC_T *
dupsv(
	REG_1 STRVEC_T *p)

DESCRIPTION

dupsv duplicates the strvec pointed to by p.

dupsv is used by the IPW image header duplication routines to duplicate strvec header values.

RETURN VALUE

pointer to the duplicate strvec, or NULL if any memory allocations fail.
