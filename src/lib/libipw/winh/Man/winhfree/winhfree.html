NAME

winhfree - free all left over parts of a WIN header

SYNOPSIS

     #include "winh.h"

int
winhfree(
	WINH_T **  winhpp,
	int        nbands)

DESCRIPTION

winhfree frees an entire winhpp structure.

RETURN VALUE

     OK for success, ERROR for failure
