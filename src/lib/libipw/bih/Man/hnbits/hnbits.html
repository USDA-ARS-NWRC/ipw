hnbits

IPW Library Function
Library - libipw
Category - bih

NAME

hnbits - number of bits in image band

SYNOPSIS

#include "bih.h"

int
hnbits(
	int             fd,	|* image file descriptor	 *|
	int             band)	|* image band #			 *|

DESCRIPTION

hnbits determines the number of bits per band in the specified band of the image currently being read/written on file descriptor fd.

hnbits may not be called before a BIH is read/written from/to fd.

RETURN VALUE

number of bits per band

GLOBALS ACCESSED
