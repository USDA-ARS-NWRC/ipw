no_history

IPW Library Function
Library - libipw
Category - bih

NAME

no_history - turn off history mechanism in bihwrite

SYNOPSIS

#include "bih.h"

void
no_history(
	int             fd)	|* image file descriptor	 *|

DESCRIPTION

no_history turns off the history mechanism. If called for file descriptor "fd" BEFORE bihwrite is called for the same file descriptor, the normal printing of the history by bihwrite will be suppressed.

no_history is normally called by programs (e.g. mux) that would otherwise produce voluminous history records

RETURN VALUE

None.
