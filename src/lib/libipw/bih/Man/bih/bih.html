IPW Library Function
Library - libipw
Category - bih

NAME

bih - get BIH header associated with file descriptor

SYNOPSIS

#include "bih.h"

BIH_T **
bih(
        int     fd)             |* image file descriptor	*|

DESCRIPTION

RETURN VALUE

pointer to the BIH header array associated with file descriptor fd; or NULL if there are no BIH headers associated with file descriptor fd.
