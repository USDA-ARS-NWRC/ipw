IPW Library Function
Library - libipw
Category - gethdrs

NAME

skiphdrs - skip IPW image headers

SYNOPSIS

#include "gethdrs.h"

void
skiphdrs(
	int      fd)		|* input image file descriptor	 *|

DESCRIPTION

skiphdrs skips (i.e. reads and discards) all IPW image headers on file descriptor fd.

skiphdrs is a simple way to skip all remaining headers on an input image, once any desired headers have been ingested.

RETURN VALUE

None.
