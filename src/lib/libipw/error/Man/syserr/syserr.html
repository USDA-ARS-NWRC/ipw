IPW Library Function
Library - libipw
Category - error

NAME

syserr - note occurrence of UNIX system error

SYNOPSIS

void syserr()

DESCRIPTION

Syserr notes the occurrence of a UNIX system error. The current value of "errno" is made accessible to the _error() error handler.

RETURN VALUE

None.

EXAMPLES

GLOBALS ACCESSED

_errno
