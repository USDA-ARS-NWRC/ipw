get_proj_units

IPW Library Function
Library - libipw
Category - mapproj

NAME

get_proj_units - get name for projection units ID

SYNOPSIS

#include "mapproj.h"

char *
get_proj_units(
	int	units_id)	|* projection units ID		 *|

DESCRIPTION

get_proj_units returns the name of the geographic projection units that corresponds to the given units ID. If an invalid ID is specified, the name "(unknown)" is returned.

RETURN VALUE

pointer to name of projection units

EXAMPLES

GLOBALS ACCESSED

None.

DIAGNOSTICS

RESTRICTIONS

The returned character string is in static memory; so do NOT attempt to "free" it up.

BUGS
