NAME

crhdup - duplicate an IPW CRH header

SYNOPSIS

#include "crh.h"

CRH_T **
crhdup(
	CRH_T **oldhpp,		|* -> old CRH array		 *|
	int     nbands)		|* # header bands		 *|

DESCRIPTION

crhdup creates a duplicate of the IPW CRH header pointed to by "oldhpp".

crhdup is an easy way for applications programs to create a new CRH (e.g., for an output image, after having read the CRH for an input image).

RETURN VALUE

A pointer to the duplicate header is returned. NULL is returned if an error occurs.
