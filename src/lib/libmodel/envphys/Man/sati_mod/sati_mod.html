Library - libmodel
Category - envphys

NAME

sati_mod - saturation vapor pressure over ice

SYNOPSIS

#include "ipw.h"
#include "envphys.h"

double
sati(
        double  tk)             /* air temperature (K)  */

DESCRIPTION

sati_mod returns the saturation vapor pressure over ice (in Pa) as a function of air temperature. Will return FALSE if an error so that the pixel can be determined.

RETURN VALUE

saturation vapor pressure (Pa).
