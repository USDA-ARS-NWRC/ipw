bowen

IPW Library Function
Library - libmodel
Category - envphys

NAME

bowen - calculates Bowen ratio (H/LE)

SYNOPSIS

#include	"ipw.h"
#include	"envphys.h"

double
bowen(
     double  p,   /* air pressure (same units as ea and es)  */
     double  ta,  /* potential air temperature (K)           */
     double  ts,  /* potential surface temperature (K)       */
     double  ea,  /* vapor pressure (same units as p and es) */
     double  es)  /* vapor pressure at surface (same units
                       as p and es)                          */

DESCRIPTION

bowen calculates the Bowen ratio (H/LE).

RETURN VALUE

Bowen ratio (H/LE)
