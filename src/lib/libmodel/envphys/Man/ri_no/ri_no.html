ri_no

IPW Library Function
Library - libmodel
Category - envphys

NAME

ri_no - bulk Richardson number at geometric mean height

SYNOPSIS

#include	"ipw.h"
#include	"envphys.h"

double
ri_no(
     double  z2,   /* upper height (m)                          */
     double  z1,   /* lower height (m) (maybe roughness length) */
     double  t2,   /* virtual temperature at z2 (K)             */
     double  t1,   /*    "        "       "  z1  "              */
     double  u2,   /* wind speed at z2 (m/s)                    */
     double  u1)   /*  "     "   "  z1   "   (may be zero)      */

DESCRIPTION

ri_no calculates the bulk Richardson number at the geometric mean height between two given heights.

RETURN VALUE

bulk Richardson number

HISTORY
