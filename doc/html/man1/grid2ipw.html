grid2ipw

IPW User Command
Category - Importing Image Data

NAME

grid2ipw - convert an Arc/Info image to an IPW image

SYNOPSIS

grid2ipw in_prefix out_image

DESCRIPTION

grid2ipw converts an image created by Arc/Info into an IPW image. This command is invoked by the AML script "grid2ipw.aml".

OPTIONS

None.

OPERANDS

in_prefix
Use in_prefix.bip and in_prefix.info as input files; these files are created by the AML script "grid2ipw.aml".

out_image
The name of the output IPW image to create.

EXAMPLES

None, since this command is invoked from an AML script.

FILES

     $IPW/aux/grid2ipw/grid2ipw.aml     AML script

DIAGNOSTICS

can't open in_prefix.bip
This input file is missing.

can't open in_prefix.info
This input file is missing.

NOTES

This command is not directly invoked by the user; rather it is called by the AML script "grid2ipw.aml".
