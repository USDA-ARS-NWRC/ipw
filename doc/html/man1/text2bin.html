text2bin

IPW User Command
Category - Importing Image Data

NAME

text2bin - convert text file with integers to binary file

SYNOPSIS

text2bin [ -1 ] [ -2 ] [ -4 ] [file ...]

DESCRIPTION

text2bin reads text integers from file (default: standard input) and writes thier binary equivalents to the standard output. If more than one file are specified, then they are read in sequence.

OPTIONS

-1
Convert input to 1-byte (char) integers.

-2
Convert input to 2-byte (short) integers.

-4
Convert input to 4-byte (long) integers.

Exactly one of -1, -2, or -4 must be specified.

OPERANDS

file
Input file to read text integers from (default: standard input).

EXAMPLES

To convert 256 text integers to an IPW lookup table: 

      text2bin -1 | mkbih -l 1 -s 256 -y 1

FILES




DIAGNOSTICS








file: bad input




	An input value could not be converted to an integer.






RESTRICTIONS




text2bin is not an IPW program, but is provided with IPW for use in
shell scripts.


Negative input values, or input values too large to be represented
in the specified number of output bytes, will be converted to
binary in a machine-dependent fashion.
