tmpt

IPW User Command
Category - Importing Image Data

NAME

tmpt - make IPW image from Thematic Mapper (TM) image on tape

SYNOPSIS

tmpt [ -b band ]

DESCRIPTION

tmpt reads a TM image off of tape and writes an IPW file.

OPTIONS

-b band
Which band to read (default: 1).

OPERANDS

None.

EXAMPLES

FILES

     The output file will be 2983 lines by 4220 samples.  It will have
     "lq" (linear quantization), "sun", and "sat" headers.

DIAGNOSTICS

RESTRICTIONS

The tape device read is /dev/nrmt0.
