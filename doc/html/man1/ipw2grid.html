ipw2grid

IPW User Command
Category - Exporting IPW Image Data

NAME

ipw2grid - convert IPW image to Arc/Info image

SYNOPSIS

ipw2grid ipw_img out_prefix

DESCRIPTION

ipw2grid converts a single-band IPW image into an Arc/Info image. This command is invoked by the AML scripts "ipw2grid.aml" and "ipw2stack.aml".

OPTIONS

None.

OPERANDS

ipw_img
Input IPW image to convert.

out_prefix
Prefix for output files: the files "out_prefix.bip" and "out_prefix.hdr" are read by the IMAGEGRID command in Arc/Info, while the file "out_prefix.lq" is read the AML script "ipw2grid.aml".

EXAMPLES

None, since this command is invoked from an AML script.

FILES

    $IPW/aux/ipw2grid/ipw2grid.aml    AML script (single-band IPW images)
    $IPW/aux/ipw2grid/ipw2stack.aml   AML script (multi-band IPW images)

DIAGNOSTICS

NOTES

This command is not directly invoked by the user; rather it is called by the AML scripts "ipw2grid.aml" and "ipw2stack.aml".
