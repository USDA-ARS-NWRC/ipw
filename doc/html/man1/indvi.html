indvi

IPW User Command
Category - Environmental Physics and Properties: Spatial

NAME

indvi - normalized difference vegetation index (NDVI)

SYNOPSIS

indvi (image)

DESCRIPTION

indvi calculates normalized difference vegetation index (ndvi) from visible and IR satellite data such as AVHRR channels 1 & 2 (see: Tucker, et al., 1986; Fung, et al., 1987): 

	{IR - VIS} / {IR + VIS}

Output is a 1-byte image between -1 and 1. Input image band 0 is Visible, and band 1 is IR.

OPTIONS

None

OPERANDS

(image)
2-band visible and IR input image

EXAMPLES

	mux vis ir | indvi

FILES

None

DIAGNOSTICS

None

RESTRICTIONS

Input image must have more than one band.
