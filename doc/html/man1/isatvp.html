isatvp

IPW User Command
Category - Environmental Physics and Properties: Spatial

NAME

isatvp - saturation vapor pressure from dew point temperature

SYNOPSIS

isatvp [ -m mask ] [image]

DESCRIPTION

isatvp is one of a group of commands designed to simulate energy, climate, or hydrologic processes or properties over an area. Most were originally written as "qdips" (Quick and Dirty Image Processing System) programs to simulate the process at a point by D. Marks, during the 1980's at UCSB. isatvp is the spatial version of satvp.

isatvp reads a single band image (default: standard input) containing dew point temperature (in degrees C) and writes the corresponding saturation vapor pressure image (in Pascals) to the standard output.

(see General Meterology, Byers, 1974)

OPTIONS

-m mask
Use mask as a mask image. Any points in mask that are 0 will have the corresponding output pixels set to 0.

OPERANDS

[image]
image input dew point temperature image

EXAMPLES

Given dew point temperature, get vapor pressure: 

	isatvp dewpt.img

FILES

     None

DIAGNOSTICS

image has bands bands; single band images only

Only single band images are accepted.

unrealistic dew point temperature: temp

A dew point temperature was given that was below 0 degrees Celcius. It is possible that a program like despike could correct these values.
