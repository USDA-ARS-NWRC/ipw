Library - libipw
Category - fpio

NAME

fpclose - close floating-point I/O file descriptor

SYNOPSIS

#include "fpio.h"

int
fpclose(
	int fd)		|* file descriptor		*|

DESCRIPTION

fpclose resets all floating-point I/O information associated with file descriptor fd, then calls pxclose to close the associated file.

Call fpclose if you need to reclaim a file descriptor that has been read|written by fpvread|fpvwrite.

Do not attempt any FP operations (including LQH access) to the file descriptor after this function is called.

RETURN VALUE

OK for success, ERROR for failure
