IPW Library Function
Library - libipw
Category - skewh

NAME

skewhwrite - write an IPW SKEWH header

SYNOPSIS

#include "file.h"

int
skewhwrite(
	int             fd,	   |* output file descriptor	 *|
	SKEWH_T       **skewhpp)   |* -> array of SKEWH pointers *|

DESCRIPTION

skewhwrite writes the array of SKEWH headers pointed to by skewhpp to file descriptor fd.

RETURN VALUE

OK for success, ERROR for failure
