IPW Library Function
Library - libipw
Category - uio

NAME

urskip - skip input on UNIX file descriptor

SYNOPSIS

#include "ipw.h"

long
urskip(
	int      fd,		|* input file descriptor	 *|
	long     nbytes)	|* #bytes to skip		 *|

DESCRIPTION

urskip causes nbytes bytes of input on fd to be skipped. nbytes must be > 0 (i.e., only forward skips are allowed).

Use urskip instead of the UNIX system call lseek to skip input data.

RETURN VALUE

>= 0 number of bytes skipped

ERROR for failure
