sunrise

IPW Library Function
Library - libmodel
Category - solar

NAME

sunrise - sunrise time (GMT) for a given place and date

SYNOPSIS

#include "ipw.h"
#include "solar.h"

datetime_t *
sunrise(
     double  lat,     /* latitude (radians)  */
     double  lon,     /* longitude (radians) */
     int     year,    /* year (e.g., 1997)   */
     int     month,   /* month (1 - 12)      */
     int     day)     /* day (1 - 31)        */

DESCRIPTION

sunrise determines the time of sunrise at a given location (specified by its latitude and longitude) and date. South latitudes and west longitudes are negative.

The returned date-time structure contains the GMT time of local sunrise. Note: it does not account for local horizons.

RETURN VALUE

non-NULL
GMT time of sunrise.

NULL
Error occurred, and an explantory message stored via the "usrerr" routine.
