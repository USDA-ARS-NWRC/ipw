Category - uio

NAME

ustdin - initialize UIO on standard input

SYNOPSIS

int
ustdin(void)

DESCRIPTION

ustdin initializes the UIO package for reading the program's standard input.

ustdin should be called by an IPW main() if the program will be reading image data from the standard input. ustdin should NOT be called if the program uses UNIX stdio input functions (e.g., scanf).

ustdin causes program termination if any errors are encountered.

RETURN VALUE

readable UNIX file descriptor
