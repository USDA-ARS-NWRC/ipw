geohfree

IPW Library Function
Library - libipw
Category - geoh

NAME

geohfree - free all left over parts of a GEO header

SYNOPSIS

     #include "geoh.h"

     int
geohfree(
     	GEOH_T **geohpp,
	int      nbands)

DESCRIPTION

geohfree frees an entire geohpp structure.

RETURN VALUE

     OK for success, ERROR for failure
