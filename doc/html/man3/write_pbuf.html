write_pbuf

IPW Library Function
Library - libimage
Category - image

NAME

write_pbuf - write unsigned-integer pixels from buffer to image

SYNOPSIS

#include "image.h"

bool_t
write_pbuf(
     IMAGE_T  *image,     /* output image              */
     PIXEL    *buffer,    /* buffer of pixels to write */
     int       nsamples)  /* # of samples in buffer    */

DESCRIPTION

write_pbuf() writes the unsigned-integer pixels in a given buffer to an image.

RETURN VALUE

TRUE
Pixels were written to the image.

FALSE
An error occurred while writing.
