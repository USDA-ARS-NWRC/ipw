evap

IPW Library Function
Library - libmodel
Category - envphys

NAME

evap - calculates evaporation rate

SYNOPSIS

#include "ipw.h"
#include "envphys.h"

double
evap(
     double  le,        /* latent heat transfer (W/m^2)  */
     double  ts)        /* surface temperature (K)       */

DESCRIPTION

evap calculates the evaporation or condensation rate from or onto a surface.

RETURN VALUE

rate of water gain or loss (kg/sec/m^2) which is also the specific discharge in mm/sec
