dew_point

IPW Library Function
Library - libmodel
Category - envphys

NAME

dew_point - dew point temperature

SYNOPSIS

#include "ipw.h"
#include "envphys.h"

double
dew_point(
     double  e)      /* vapor pressure (Pa) */

DESCRIPTION

dew_point returns the dew point temperature for a given vapor pressure.

RETURN VALUE

dew point temperature (K)

HISTORY

Apr 1989
Fixed to accept saturation over ice. D. Marks, USEPA, Corvallis, OR
