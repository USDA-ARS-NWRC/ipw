IPW Library Function
Library - libipw
Category - uio

NAME

uputs - write a string to a UNIX file descriptor

SYNOPSIS

#include "ipw.h"

int
uputs(
	int             fd,	|* input file descriptor	 *|
	const char     *buf)	|* -> input buffer		 *|

DESCRIPTION

uputs writes an EOS-terminated string from buf to fd. The trailing EOS is not written.

Use uputs for efficient output of ASCII text (e.g., image headers) to IPW data files.

RETURN VALUE

>= 0 number of characters written

ERROR errors or end-of-file.
