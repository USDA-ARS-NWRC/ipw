yday2mday

IPW Library Function
Library - libipw
Category - datetime

NAME

yday2mday - convert yearday into month and day

SYNOPSIS

#include "ipw.h"

void
yday2mday(
	int 	year,		|* e.g., 1997		*|
	int     yrday,		|* yearday [1 - 366]	*|
	int   * month,		|* output: [1 - 12]	*|
	int   * day)		|* output: [1 - 31]	*|

DESCRIPTION

yday2mday converts a given yearday into the month and day for the given year.

RETURN VALUE

None.
