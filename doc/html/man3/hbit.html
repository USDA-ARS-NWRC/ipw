NAME

hbit - find highest-order bit in integer

SYNOPSIS

#include "ipw.h"

int
hbit(
	unsigned        i)	|* unsigned integer	*|

DESCRIPTION

hbit returns the highest bit set in the unsigned argument passed it and returns that bit. A return value of zero indicates the value passed is zero.

hbit is normally called to find the number of bits needed to hold an integer.
