NAME

walksv - step through strings in strvec

SYNOPSIS

#include "ipw.h"

char *
walksv(
	REG_1 STRVEC_T *p,
	bool_t          reset)

DESCRIPTION

walksv sequentially accesses the strings in strvec p. If reset is true, then the first string is accessed. If reset is false, then the next string is accessed.

Calling addsv or delsv may confuse walksv's notion of what the "next" string is. Calling walksv with reset == TRUE avoids this problem.

RETURN VALUE

pointer to string in p.
