IPW Library Function
Library - libipw
Category - geoh

NAME

geohwrite - write an IPW GEOH header

SYNOPSIS

#include "file.h"

int
geohwrite(
	int             fd,	  |* output file descriptor	 *|
	GEOH_T        **geohpp)	  |* -> array of GEOH pointers	 *|

DESCRIPTION

     geohwrite writes the array of GEOH headers pointed to by geohpp

to file descriptor fd.

geohwrite is called by IPW application programs to write GEOH headers.

RETURN VALUE

OK for success, ERROR for failure
