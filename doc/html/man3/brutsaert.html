brutsaert

IPW Library Function
Library - libmodel
Category - radiation

NAME

brutsaert - calculates atmospheric emissivity

SYNOPSIS

#include "ipw.h"
#include "radiation.h"

double 
brutsaert(
     double  ta,       /* air temp (K)                   */
     double  lambda,   /* temperature lapse rate (deg/m) */
     double  ea,       /* vapor pressure (Pa)            */
     double  z,        /* elevation (z)                  */
     double  pa)       /* air pressure (Pa)              */

DESCRIPTION

brutsaert calculates atmospheric emissivity using a modified form of the equations by W. Brutsaert (see the reference below).

RETURN VALUE

atmospheric emissivity
