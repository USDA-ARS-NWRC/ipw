IPW Library Function
Library - libipw
Category - orh

NAME

orhwrite - write an IPW ORH header

SYNOPSIS

#include "file.h"

int
orhwrite(
	int       fd,		|* output file descriptor	 *|
	ORH_T   **orhpp)	|* -> array of ORH pointers	 *|

DESCRIPTION

orhwrite writes the array of ORH headers pointed to by orhpp to file descriptor fd.

RETURN VALUE

OK for success, ERROR for failure
