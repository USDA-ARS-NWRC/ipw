IPW Library Function
Library - libipw
Category - error

NAME

uferr - note occurrence of UNIX file error

SYNOPSIS

void uferr(fd)
int fd;

DESCRIPTION

Uferr notes the occurrence of an error on the UNIX file descriptor fd. The _error() error handler uses this information to print the name of the file involved.

RETURN VALUE

None.
