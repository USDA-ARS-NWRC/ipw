NAME

read_proj - read projection definition file

SYNOPSIS

#include "mapproj.h"

struct projdef *
read_proj(
	char	*filename)	|* name of projection def file	 *|

DESCRIPTION

read_proj reads the given projection definition file, writes the contents into a projdef struct, and returns a pointer to the struct. The projection definition files are ASCII files with the following format: 

	projection ID
	units code
	zone code
	datum code
	projection parameters (15 values)

RETURN VALUE

non-NULL pointer to allocated & initialized projdef struct

NULL error occurred; message stored via 'usrerr'
