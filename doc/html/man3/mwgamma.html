IPW Library Function
Library - libmodel
Category - radiation

NAME

mwgamma - gamma values for twostream calculations

SYNOPSIS

#include "ipw.h"
#include "radiation.h"

void
mwgamma(
     double   u0,       /* cosine of illumination angle     */
     double   w,        /* single-scattering albedo         */
     double   g,        /* scattering asymmetry parameter   */
     double  *gam,      /* output gamma values (4)          */
     int      method)   /* DELTA_EDDINGTON or MEADOR_WEAVER */

DESCRIPTION

mwgamma computes the gamma values for twostream calculations. The method used to compute the gamma values is specified by one of two constants defined in the "radiation.h" header file: DELTA_EDDINGTON or MEADOR_WEAVER.

RETURN VALUE

None.
